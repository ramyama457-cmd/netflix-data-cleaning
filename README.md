# netflix-data-cleaning

## Description
This project cleans the Netflix dataset by handling missing values, removing duplicates, and saving a cleaned version.

## Steps Performed
- Loaded the dataset using pandas
- Checked and filled missing values
- Removed duplicates
- Exported final cleaned dataset as fully_cleaned_netflix_titles.csv

## Tools Used
- Python
- Pandas
- NumPy

## How to Run
```bash
python data_cleaning.py
